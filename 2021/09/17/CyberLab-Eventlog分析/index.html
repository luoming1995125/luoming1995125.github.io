事件調查思路

  • 人
    • 確認誰被攻擊
    • 確認攻擊來源
  • 事
    • 確認事件類別
    • 過去是否發生過
  • 時
    • 確認攻擊時間點
  • 地
    • 確認發生的網段
    • 誰可以存取
    • 平時如何管理
  • 物
    • 確認有甚麼東西產生
    • 確認是否有東西流出

linux forensic (file-webshell)

  • Web service access logs:
    • mtime n : File’s data was last modified n minutes ago
    • atime n : File was last accessed n minutes ago.
  • ps -ef
  • ls /proc
  • strace -f -p pid
  • lsof -i:port
  • lsof -p pid
  • crontab -e
  • cat /etc/rc.local
  • ls /etc/init.d
  • ls /etc/profile.d
  • ls /var/log/cron*
  • ls /var/log/secure*
